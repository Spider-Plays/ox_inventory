# ox_inventory (Spider Fork)

Fork of `ox_inventory` with custom inventory UX tweaks, runtime UI sizing config, and Discord logging.

> **This is a modified fork of [ox_inventory](https://github.com/overextended/ox_inventory) with a redesigned UI to match the visual style of sd-crafting.**
